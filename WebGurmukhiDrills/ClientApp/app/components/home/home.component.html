Overview

This site will aid you in your personal or private studies of reading and writing the gurmukhi language via english, but other good resources exist which help you learn. If you are brand new to Gurmukhi, then check out the starting point at the foot of this page. There are plenty of other resources which will teach you how to speak punjabi.

This site is split into learning the gurmukhi letters and common vocabulary. If you are a complete beginner then I will recommend the that you see below for links for other good resources.

Please note this site uses newer javascript technology, which may not be supported on older browsers, mobiles and tables.

Letter Board

The Letter Board is a good overview of the letter groups and gives a review of a section of letters.

Letter Slide

The Letter Slide displays the letters is a slider panel and automatically sounds the pronunciation and name of letters.

Multiple Choice

The Multiple Choice will create multiple choice question for the given letter groupings.

Random Words

The Random Words will create random letter requiring the anwswers in english or gurmukhi for the given letter groupings. This is so the user can get an understanding of translating gurmukhi via english.

Word List

The Word List display a list of words in a selected cateogry.

Word Slides

The Word Slides display a word slides in a selected cateogry.

Word Mulitple Choice

The Word Mulitple Choice create a mulitple choice of words where you need to match the translation.

Word Match

The Word Match creates question of words where you need to match the translation in gurmukhi and english.
